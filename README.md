# Advent of Code Solutions

This repository contains my solutions for Advent of Code challenges.

## Structure

- `2025/` - Solutions for Advent of Code 2025
  - `Day 1/` - Day 1 solutions
  - `Day 2/` - Day 2 solutions  
  - `Day 3/` - Day 3 solutions

## Running Solutions

Each day's folder contains:
- C++ source files (`firstTask.cpp`, `secondTask.cpp`)
- Compiled executables (`firstTask`, `secondTask`)
- Input files (`input.txt`, etc.)

To compile and run:
```bash
cd 2025/Day\ X/
g++ -o firstTask firstTask.cpp
./firstTask
```

## Language

Solutions are primarily written in C++.
